# ClayBytes.github.io

ClayBytes is a senior IT consultancy based in the Netherlands, specializing in enterprise engineering, fullstack development, and AI‑driven automation. This repository contains the source code for **ClayBytes.nl**, the public website that represents the consultancy and its long‑term mission.

## About ClayBytes
ClayBytes is the professional home of **Clayton Reitz**, a senior fullstack engineer and agentic AI architect with more than 15 years of experience across backend systems, modern frontends, and intelligent automation. The consultancy delivers high‑performance engineering for Dutch MKB clients and international partners.

ClayBytes is also the foundation for a larger vision:  
**ITDojo — the Skill Factory.**  
A global ecosystem where technologists can learn, build, collaborate, and future‑proof their skills.  
The consultancy phase serves as the stepping stone toward building this network of connected technical hubs.

## Vision: The Skill Factory
“Building the future of IT, one gizmo at a time.”

The ITDojo concept introduces:
- Hands‑on learning zones  
- Robotics and hardware labs  
- AI workstations  
- Workshops and mentorship  
- A global Portal Network connecting Dojos worldwide  

ClayBytes provides the financial and technical foundation for this long‑term mission.

## Website Features
The ClayBytes website is designed to be clean, fast, and future‑ready:
- Responsive layout  
- Light/dark theme with smooth transitions

- Semantic HTML  
- HubSpot integration for secure contact forms  
- Clear navigation and accessible design

## Development
To run or modify the site locally:
1. Clone the repository  
2. Open `index.html` in your browser  
3. Edit HTML/CSS/JS as needed  
4. Push changes to deploy automatically via GitHub Pages
